About Run Boaster

Designed to give runners a place to analayse and share running data.

Compare runs

Once a GPX file is uploaded we calculate a summary. The summary shows each kilometer and the pace per kilometer, as well as the total time and distance.

Runners can then see how their runs compare using the table, map or charts.

Tech Stack

Run Boaster is a single page application built using MongoDb, Expressjs, Angularjs and Nodejs, AKA the MEAN stack.

Leafletjs is used to display maps.

D3 is used to display charts.

The GeoJson format is used to store runs.
