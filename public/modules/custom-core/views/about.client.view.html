What in the world is Run Boaster?

Inspired by a brother's need to boast and brag about their running exploits we thought we'd create the perfect boasting platform.

The hope is that one day we'll be able to boast about our own running. But we're still in the slow lane for now.

How it works

Record your runs using the many apps and devices

Download your data as GPX files. Then upload them to Run Boaster

Compare your runs with our charts, maps and tables, then brag about them to friends

We're under active (no pun intended) development

Yes we're still developing the site. So what you see is the initial stages of what we hope to achieve.

We'll continue to update the site with new functionality.

Our aim is to include sharing abilities based around smug points.

We'll compare your latest run against your others and optionally against peers to tell you if you've run like a granny or like Rambo.

Then you can choose to share it with the world via your usual social channels.

Tech Stack

Run Boaster is a single page application built using MongoDb, Expressjs, Angularjs and Nodejs, AKA the MEAN stack.

Leafletjs is used to display maps.

D3 is used to display charts.

The GeoJson format is used to store runs.
